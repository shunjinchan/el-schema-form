## 安装

```shell
yarn add git@github.com:shunjinchan/el-schema-form.git
```

### 引入 el-schema-form

```javascript
import ElSchemaForm from 'el-schema-form'
import 'el-schema-form/dist/el-schema-form.css'

Vue.use(ElSchemaForm)
```
